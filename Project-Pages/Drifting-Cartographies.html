Drifting
Cartographies
This project explores the intersection of data visualization and systemic thinking. Through a series of interactive diagrams and visual models, it examines how complex systems can be represented in ways that are both informative and aesthetically engaging.

The work draws on principles from network theory, information design, and cognitive science to create visualizations that reveal hidden patterns and relationships. Each visualization is designed to be both a tool for analysis and an object of contemplation.

The project includes ten distinct visualizations, each focusing on a different aspect of systemic relationships - from social networks to ecological systems to information flows.
